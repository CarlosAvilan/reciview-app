## Descripción del Proyecto
Aplicación móvil desarrollada en **Kotlin** y **Jetpack Compose** para la gestión inteligente de gastos personales. La app permite escanear tickets mediante OCR, categorizar consumos y sincronizar datos con un backend en **Ktor** y **MongoDB**.

## Tecnologías Utilizadas
* **Frontend:** Kotlin, Jetpack Compose, CameraX, ML Kit.
* **Backend:** Ktor, MongoDB Atlas.
* **Arquitectura:** MVVM + Repository Pattern.
